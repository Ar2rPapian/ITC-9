The requarements of the task - Check the validity of json file

There are the folowing functions:
    - inputJsonStringInFile() 
        the function writes json string in the file

    - getJsonStringFromFile()
        the function returns the string of the json

    - checkBrackets(jsonString)
        the function takes the string of the json as argument
        checks the existance of the brackets
        gives True or False

    - checkCountKeyValueList(keyList,valueList)
        the function takes the lists of keys and values as argument
        checks the lengths of lists of keys and valus
        gives True or False

    - checkKey(keyList)
        the function takes the list of key as argument
        checks the keys are repeating or not
        gives True or False
    - checkRepeatKey(keyList)
        the function takes the list of keys as argument
        checks the validity of the key
        gives True or False
    
    - checkValue(valueList)
        the function takes the list of values as argument
        checks the validity of the value
        gives True or False
    
    - getKeysAndValues(jsonString,key,value)
        the function takes the json string and the lists of key and values as argument
        gives lists of keys and values
